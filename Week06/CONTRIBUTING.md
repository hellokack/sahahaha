# 🤝 기여 가이드라인 (Contributing Guide)

사하구청 AI 상담사 오픈소스 프로젝트에 오신 것을 환영합니다!
누구나 버그 수정, 기능 추가, 문서 개선에 참여할 수 있습니다.

## 🛠️ 기여 방법
1. 이 저장소를 Fork 합니다.
2. 새로운 브랜치를 생성합니다. (`git checkout -b feature/새기능`)
3. 코드를 수정하고 커밋합니다. (`git commit -m "feat: 새로운 기능 추가"`)
4. 브랜치에 푸시합니다. (`git push origin feature/새기능`)
5. Pull Request (PR)를 생성해 주세요! 팀원들의 리뷰 후 병합됩니다.
